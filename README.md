# Project đồ án 2 - Mô phỏng con lắc ngược trên Simscape và thiết kế bộ điều khiển EC+LQR
## Hướng dẫn sử dụng
1. Download về máy tính
2. Chạy file "parameters.m" trên MATLAB, để workspace ghi nhận các biến
3. Chạy file "test_inverted_pendulum" để chạy mô hình.
## Mục tiêu:
- Tìm hiểu về mô hình con lắc ngược, cũng như cách mô hình hóa, mô phỏng
- Tìm hiểu và mô phỏng LQR cho việc giữ cân bằng của con lắc ngược
- Tìm hiểu và mô phỏng bộ điều khiển năng lượng (Energy Control - EC) dùng cho việc đưa con lắc ngược từ vị trí cân bằng bền (phía dưới) lên vị trí cân bằng không bền (phía trên)
